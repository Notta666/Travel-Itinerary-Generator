# Contributing to Travel-Itinerary-Generator

Thank you for your interest in contributing! We welcome bug reports, feature requests, and pull requests.

## How to Contribute

### 1. Reporting Issues
- Check the issues tab first to make sure it hasn't been reported yet.
- Provide a clear, concise description of the issue and steps to reproduce.

### 2. Submitting Pull Requests
- Fork the repository.
- Create a new branch for your edits: `git checkout -b feature-or-bugfix-name`.
- Ensure the code conforms to standard formatting and passes all linting.
- Open a PR describing your changes.

### 3. Setup Development Environment
```bash
# Clone the repository
git clone https://github.com/Notta666/Travel-Itinerary-Generator.git
cd Travel-Itinerary-Generator

# Install development requirements
python -m pip install -r requirements.txt
```

### 4. Code Style Guidelines
- Follow standard Python styling guidelines (PEP 8).
- Use descriptive naming conventions.
- Keep comments and docstrings up to date.
